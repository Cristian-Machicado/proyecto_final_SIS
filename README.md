# Sistema de Perfil Universitario - SIS-500
[cite_start]Aplicación web desarrollada como proyecto final del curso de Programación Avanzada en la Universidad Privada San Francisco de Asís (USFA)[cite: 88, 90, 91, 113].

## Tecnologías utilizadas
* [cite_start]PHP 8.x [cite: 93]
* [cite_start]Laravel 11.x 
* [cite_start]CSS3 (Diseño propio, sin frameworks externos) [cite: 95]
* [cite_start]MySQL [cite: 95]
* [cite_start]GIT + GitHub [cite: 96]

## Funcionalidades
* [cite_start]Página de perfil personal del estudiante [cite: 98]
* [cite_start]Listado de materias con notas y estados [cite: 98]
* [cite_start]Formulario de contacto con validación PHP [cite: 98]
* Controladores para gestión de páginas (`PaginaController`)
* Sistema de vistas dinámicas con Blade (`inicio`, `materias`, `contacto`, `sobre-mi`)
